Написать класс LimitedStack, который реализует стек с ограничением размера. Этот класс должен использовать дженерики для
обеспечения обобщенности. Класс должен иметь следующие методы:
push(E element): добавляет элемент на вершину стека. Если стек уже заполнен (достиг максимального размера), то
новый элемент не добавляется и выводится сообщение об ошибке.
pop(): удаляет элемент из вершины стека и возвращает его. Если стек пуст, выводится сообщение об ошибке.
peek(): возвращает элемент, находящийся на вершине стека, при это не удаляя его. Если стек пуст, выводится сообщение об ошибке.
isEmpty(): возвращает true, если стек пуст, иначе false.
isFull(): возвращает true, есди стек достиг максимального размера, иначе false.
